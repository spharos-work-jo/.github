##  💁 간단 소개

### "포인트 적립 및 선물, 쿠폰, 이벤트 등을 제공하는 포인트 서비스"
휴대폰 인증을 하고 가입한 유저가 포인트 적립, 다른 유저에게 선물을 할 수 있고 쿠폰 다운로드를 할 수 있는 서비스


- 🔗 LINK : https://ssg-point.vercel.app
- 🎼 Period : 2023.07.31 ~ 2023.09.18 (8주간 진행)

## 역할
✨김빛누리(조장) :     
-백엔드: 유저, 쿠폰, 매장 
-프론트: 지도로 매장 조회  
-인프라

✨김성용 :  
-백엔드 : 이벤트, 포인트 조회, 포인트 선물, 포인트 적립    

✨고신영 :  
-프론트 : 포인트, 유저  
-서기  

✨김형진 :  
-프론트 : 이벤트, 쿠폰, 메인 홈, 로그인    

✨배송윤 :  
-프론트 : 마이 라운지, 사이드 메뉴바
